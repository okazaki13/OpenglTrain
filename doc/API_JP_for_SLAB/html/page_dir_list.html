本ソフトウェアのディレクトリ構造について 

AHS2009
├─bin                         実行用DLL・サンプルプログラムの実行バッチファイル集
├─data                        設定ファイル(.xml)
├─Debug                       デバック用DLLなど（将来削除予定）
├─doc                         ドキュメント関係
│  ├─API_EN                  本ソフトウェアのマニュアル（英語版・未完成）
│  │  └─html
│  ├─API_JP                  本ソフトウェアのマニュアル（日本語版）
│  │  └─html
│  └─ode                     ODEのマニュアル
│      └─docs
├─Driver                      デバイスドライバ
├─include                     インクルードファイルのディレクトリ
│  ├─AHSAPI
│  ├─drawstuff
│  ├─glut
│  ├─HbmsAPI
│  ├─ode
│  ├─SpidarAHS               <- 本ソフトウェアのインクルードファイル
│  └─SpidarAPI
├─lib                         ライブラリファイルのディレクトリ
├─license                     ライセンスファイルのディレクトリ
│  ├─AquesTalk
│  ├─ode
│  └─xerces
├─Release                     リリース用DLLなど（将来削除予定）
├─Sample                      サンプルプログラム集
│  ├─Spidar4_Ex01            SPIDARの実行（基本編）
│  │  ├─Debug
│  │  └─Release
│  ├─Spidar4_Ex02            SPIDARの実行（glut編）
│  │  ├─Debug
│  │  └─Release
│  ├─Spidar4_Ex03            SPIDARの実行（ODE編）
│  │  ├─Debug
│  │  └─Release
│  ├─SpidarG_Ex01            SPIDAR-Gの実行（基本編）
│  │  ├─Debug
│  │  └─Release
│  ├─SpidarG_Ex02            SPIDAR-Gの実行（glut編）
│  │  ├─Debug
│  │  └─Release
│  ├─SpidarG_Ex03            SPIDAR-Gの実行（ODE編）
│  │  ├─Debug
│  │  └─Release
│  └─SpidarX_Ex01            ワイヤ単位の制御用サンプルその１
│      ├─Debug
│      └─Release
│  └─SpidarX_Ex02            ワイヤ単位の制御用サンプルその２
│      ├─Debug
│      └─Release
└─textures                    テクスチャなどの素材集（ODE）

※　注　　意　※
dataディレクトリ下にある設定ファイルは起動時に必要となる重要なファイルです。
変更する場合は特に注意してください。
また、設定ファイル（.xml）の書式を決定する文書型定義(.dtd)は隠しファイルとして同ディレクトリに格納されています。 絶対変更しないでください。
