サンプルプログラムについて

サンプルプログラムをいくつか用意しています． サンプルプログラムの実行形式をbinディレクトリにソースファイルをSampleディレクトリに収録しています．

サンプルプログラムの実行形式

binディレクトリには，Spidar,SpidarGのサンプルプログラムの実行形式(.batファイル)がフレームの形状に合わせて用意されています．

例えば，SpidarGCCフレームを用いて物理シミュレータODEのサンプルプログラムを実行したければ SpidarGCC_ode_test.bat を起動すれば実行できます．

SpidarGCCフレームを用いてSpidarG_ode_test.batを実行した場合， フレームによってモータの配置や回転方向が異なるため，正しく動作しません．
必ずフレームに合わせた.batファイルを起動してください．



サンプルプログラムのリストと説明

AHS_TEST_for_G.bat Ｘ型フレーム用のテストプログラム
AHS_TEST_for_GCC.bat GCCフレーム用のテストプログラム
SpidarGCC_ode_test.bat SpidarGCCフレームで物理シミュレータODEのサンプルプログラムを実行する
SpidarGCC_open_gl_test.bat SpidarGCCフレームでOpenGLのサンプルプログラムを実行する
SpidarGCC_simple_test.bat SpidarGCCフレームで基本的なコンソールのサンプルプログラムを実行する
SpidarG_ode_test.bat SpidarGをX型フレームで物理シミュレータODEのサンプルプログラムを実行する
SpidarG_open_gl_test.bat SpidarGをX型フレームでOpenGLのサンプルプログラムを実行する
SpidarG_simple_test.bat SpidarGをX型フレームで基本的なコンソールのサンプルプログラムを実行する




サンプルプログラムのソースコード

・Sampleディレクトリ下にSpidar、SpidarGのサンプルプログラムを収録しています。
・Sample.slnを開くとVisualStudioC++が起動し、いくつかのサンプルプログラムが実行可能になります。
・計算機が高速でないとなめらかな力覚提示が困難になる可能性があります。
・VR世界の規模などに応じ、計算機は十分高速なものを使用してください。
