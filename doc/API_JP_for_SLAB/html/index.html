SpidarAHS APIマニュアル

1.0AforSLAB

Introduction

本システムは、まだ不完全ですが、新しいコントローラ用のSpidarAHS(SpidarAHS.h)APIライブラリです．
これから徐々に充実していく予定です．APIライブラリはファイル名の後ろに日付でバージョンアップしていきます．
現在はWindows対応のみ（Windows XP, Windows Vista, Windows 7）です。
※英語版のマニュアルは未完成です。

※　注　意　※
佐藤研内での利用において制限なしで利用できます．
外部での無断使用は禁止します．使用したい場合は連絡してください． 現状非公開です。

本ソフトウェアはワイヤ駆動型力覚提示装置を制御するためのライブラリです。
本ライブラリでは
・4本糸で並進3自由度の位置計測および力覚提示可能なデバイス
・8本糸で並進・回転6自由度の位置・姿勢計測および力覚提示可能なデバイス
を制御できます。

仕様

ライブラリ形式　　　　　 　 Win32 DLL(ダイナミックリンクライブラリ)　　　　　　
動作OS Windows7(64bit, 32bit), WindowsVista(64bit, 32bit), WindowsXP(32bit), Windows2000(32bit)
開発ツール Microsoft Visual C++ 2008
開発言語 C++

※DLLに関する詳しい情報はMicrosoftの情報に従ってください。

Spidarを利用するには、下記のファイルを用いることで可能となります。 （ファイル名(*.h)をクリックするとAPIのリファレンスマニュアルにリンクします。）

ヘッダーファイル 　　　　　 SpidarAHS.h　　　　　　　　　　　　　　　　　　　　　　
ライブラリファイル SpidarAHS.lib
DLLファイル SpidarAHS.dll

このライブラリにより、C++の関数からSpidarを制御できます。
これまでのSpidarライブラリでは力覚提示の制御ループはライブラリに含まれませんでしたが、 本ライブラリでは力覚提示の制御ループをライブラリ内部でマルチメディアタイマーを用いて実現しています。 より簡単にSpidarを実行することが可能になりました。
