PATHの設定 WindowsXP/2000の場合

本ソフトウェアではシステムを実行するのに環境変数PATHの設定が必要です。
環境変数PATHに本ソフトウェアのディレクトリであるAHS2009¥binを追加する必要があります。

Step 1:デスクトップ上のアイコン「マイコンピュータ」を右クリックします。ポップアップメニューの中から「プロパティ」を選択します。

もしくは「コントロールパネル」から「システム」を実行します。

Step 2:「システム」の左側にある「タスク」の中の「システムの詳細設定」をクリックします。

Step 3:「環境変数」ダイアログ上のリストの中の「システム環境変数」内に「Path」がありますのでクリックして編集ボタンを押します。

※現在ログイン中のユーザーアカウントでのみ本ソフトウェアを利用する場合、 および、ユーザーアカウントが「コンピュータの管理者」でない場合には、 「ユーザー環境変数」の「新規」（Pathが存在していない場合）か 「編集」（Pathがすでに存在している場合）のボタンをクリックし、 次の3.の要領で「Path」を追加してください。

Step 4:「システムの変数の編集」ダイアログ内の「変数値」の末尾に次の一文を追加します。

例：本ソフトウェアライブラリAHS2009をCドライブのトップ(C:¥AHS2009)にインストール（コピー）した場合

;C:¥AHS2009¥bin

本ソフトウェアをインストール（コピー）した場所が上記と異なる場合はその場所に合わせて設定してください。

この際、文字はすべて半角英数です。 冒頭の「;」を忘れないように注意してください。 すでに入力されている値は絶対に消したり、書き換えたりしないでください。 システム起動エラーを起こすことがあります。

Step 5:作業が終了したら「OKボタン」をクリックします。後は、それぞれ「システム」ダイアログと「システムのプロパティ」の「OKボタン」をクリックした後、Windowsを再起動して作業終了です。
