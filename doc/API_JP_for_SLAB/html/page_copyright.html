著作権について

音楽、映像、コンピュータ・プログラム、データベースなどは著作権法により、その著作者および著作権者の権利が保護されています。 こうした著作物を複製することは、個人的にまたは家庭内で使用する目的でのみ行うことができます。 上記の目的を超えて、権利者の了解なくこれを複製(データ形式の変換を含む)、改変、複製物の譲渡、ネットワーク上での配信などを行うと、「著作権侵害」「著作者人格権侵害」として損害賠償の請求や刑事処罰を受けることがあります。 本製品を使用して複製などをする場合には、著作権法を厳守の上、適切な使用を心がけてください。
